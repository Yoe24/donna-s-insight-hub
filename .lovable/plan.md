

# Donna — Assistant IA pour Avocats

## 1. Landing Page (style Harvey.ai)
- Design épuré noir & blanc, typographie serif élégante
- Logo "Donna" en haut à gauche
- Navigation minimaliste (Plateforme, À propos, Se connecter)
- Hero section : grand slogan accrocheur + sous-texte expliquant la valeur pour les avocats
- Bouton CTA vert "Commencer là" bien visible
- Fond blanc, texte noir, ambiance premium/juridique

## 2. Page Tutoriel "Connecter votre boîte mail"
- Page blanche épurée avec étapes numérotées (1, 2, 3)
- Explication simple de la connexion Gmail → Donna
- Bouton "C'est fait" pour confirmer et accéder au dashboard
- Ton rassurant, adapté à un public non-technique

## 3. Dashboard (inspiré Sierra)
### Layout
- Sidebar gauche avec navigation (style Sierra) : Tableau de bord, Configuration, Dossiers
- Zone principale avec les KPIs et le flux d'activité

### KPIs en haut (cartes)
- **Mails reçus** (nombre + tendance)
- **Mails ouverts** (nombre + tendance)
- **Brouillons créés** (nombre + tendance)
- **Brouillons validés** (nombre + tendance)

### Section ROI
- **Temps gagné aujourd'hui** (en heures/minutes)
- **Argent économisé** (basé sur le taux horaire configurable)

### Flux d'activité en temps réel (zone principale)
- Liste type boîte mail : pour chaque email →
  - Résumé concis du contenu (pas besoin d'ouvrir)
  - Expéditeur, heure de réception
  - Statut (brouillon généré, en attente…)
  - Bouton "Copier le brouillon" à droite
- Scroll infini, trié par date

## 4. Page Configuration (sidebar)
- Informations de l'avocate (nom, cabinet, spécialité)
- Exemples de mails pour capter le ton
- Signature email
- Taux horaire (pour le calcul ROI)

## 5. Page Dossiers (sidebar)
- Liste des dossiers clients créés automatiquement à partir des emails
- Pour chaque dossier : nom du client, nombre de mails, date du dernier mail
- Indication de la rétention (1 semaine, 2 semaines, 1 mois)
- Vue épurée en cartes ou tableau

## Design & Couleurs
- Palette Harvey.ai : fond blanc, texte noir, accents verts subtils (inspirés Sierra pour le dashboard)
- Typographie serif pour les titres (élégant/juridique), sans-serif pour le corps
- Coins arrondis doux, ombres légères, beaucoup d'espace blanc
- Données fictives/mock pour tous les KPIs et le flux d'activité (pas de backend pour l'instant)
